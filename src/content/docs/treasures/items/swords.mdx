**Волшебные мечи**

| ----- | ----- | ----------------------------------------- |
| 1     | 01–02 | Меч -1, проклятый                         |
|       | 03–04 | Меч -2, проклятый                         |
| 2     | 05–44 | Меч +1                                    |
| 3     | 45–50 | Меч +1 (+2 против оборотней)              |
| 4     | 51–56 | Меч +1 (+2 против магов)                  |
| 5     | 57–61 | Меч +1 (+3 против драконов)               |
|       | 62–66 | Меч +1 (+3 против зачарованных существ)   |
|       | 67–71 | Меч +1 (+3 против существ с регенерацией) |
| 6     | 72–76 | Меч +1 (+3 против нежити)                 |
|       | 77    | Истощающий силы меч +1                    |
|       | 78–81 | Пылающий меч +1                           |
| 7     | 82–89 | Сияющий меч +1                            |
|       | 90–92 | Меч искателя +1                           |
|       | 93    | Меч исполнения желаний +1                 |
| 8     | 94–96 | Меч +2                                    |
|       | 97–98 | Меч очарования +2                         |
|       | 99–00 | Меч +3                                    |

**B:** обычные персонажи (уровни 1–3).\
**X:** опытные персонажи (уровни 4+).

**Использование:** согласно стандартным ограничениям класса персонажа.

## Проклятые мечи

**Штраф:** для проклятых мечей указаны значения штрафов к проверкам атаки и урона.

**Невозможность избавиться:** однажды применив проклятый меч в бою, персонаж попадает под действие злых чар и уже не может от него избавиться. Напротив, он будет предпочитать этот меч всем остальным и стараться пользоваться им как можно чаще. Снять проклятие можно только с помощью магии.

## Зачарованные мечи

**Бонус:** для зачарованных мечей указаны значения бонусов к проверкам атаки и урона.

**Бонус против типа существ**: некоторые мечи имеют отдельный бонус к атакам и урону против существ определённого типа.

**Бонус против зачарованных существ:** зачарованными существами считаются монстры, созданные, оживлённые или призванные с помощью магии.

## Разумные мечи

Некоторые волшебные мечи, наряду с другими особенностями, обладают собственным разумом и характером. Для того чтобы определить возможности разумного меча, см. раздел [«Разумные мечи»](/treasures/items/sentient-swords).

**Вероятность:** если [рефери](/adventures/running-adventures/the-referees-role) хочет случайным образом определить, обладает ли волшебный меч собственным разумом, то вероятность этого равна 30%.

## Мечи с особым предназначением

Некоторые волшебные мечи наделены особым предназначением. Эти мечи всегда разумны и необычайно могущественны. Для того чтобы определить возможности меча с особым предназначением, см. раздел [«Разумные мечи»](/treasures/items/sentient-swords).

**Вероятность:** лишь 1 из 20 волшебных мечей обладает особым предназначением. При желании рефери может не определять это случайным образом, а просто разместить на карте мечи с особым предназначением в соответствии со своим замыслом.

## Истощающий силы меч +1

Истощает жизненную силу своих жертв.

- **При попадании:** владелец меча может приказать ему истощить энергию цели.

- **Истощение силы:** помимо получения обычного урона, цель навсегда лишается одного уровня опыта (или кости здоровья). При этом теряется не только кость или пункты здоровья, но и все остальные связанные с данным уровнем преимущества (например, [заклинания](/magic/magic/spells), [бонусы к испытаниям](/adventures/adventuring/saving-throws) и т. д.) Значение ПО персонажа возвращается к минимальному для текущего уровня. Если персонаж потерял все уровни опыта, то он умирает.

- **Заряды:** всего меч может истощить количество сил противника, эквивалентное 1d4+4 уровням. После этого он станет обычным волшебным мечом +1.

## Пылающий меч +1

По приказу владельца меч начинает пылать языками пламени. Пылая, меч:

- **Излучает свет:** в радиусе 30 футов.

- **Может поджигать предметы:** может использоваться аналогично факелу, чтобы поджигать различные объекты (например, [паутину](/magic/arcane/arcane-spells-2#паутина)).

- **Наносит урон огнём:** это может оказать особое воздействие на некоторых существ.

- **Даёт преимущество в атаке:** против некоторых типов существ:

  - +2 против [троллей](#) (и других существ, которые не могут восстанавливать ПЗ, утраченные от урона огнём).

  - +2 против [пегасов](#), [гиппогрифов](#), [птиц рух](#) (и других пернатых).

  - +3 против [энтов](#) (и других растительных существ).

  - +3 против [нежити](#).

- **Продолжительность действия:** меч прекращает пылать только по приказу его владельца.

## Сияющий меч +1

По приказу владельца источает свет в радиусе 30 футов.

## Меч очарования +2

Дарует своему владельцу силу очаровывать других людей.

- **Воздействие:** действует аналогично заклинанию [«Очарование личности»](/magic/arcane/arcane-spells-1#очарование-личности).

- **Частота использования:** особую способность меча можно применять не чаще трёх раз в неделю.

## Меч искателя +1

Позволяет находить предметы в радиусе 120 футов.

- **Воздействие:** действует аналогично заклинанию [«Нахождение объекта»](/magic/arcane/arcane-spells-2#нахождение-объекта).

- **Частота использования:** можно применять до одного раза в день.

## Меч исполнения желаний +1

Обладает силой исполнять [желания](/treasures/treasures/magic-items#желания) своего владельца.

- **Заряды:** меч может исполнить 1d4 желаний.

- **Исполнение:** владелец меча должен проговорить своё желание вслух, чтобы оно исполнилось.
