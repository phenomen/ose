## Самоцветы

Ценность каждого самоцвета в кладе определяется с помощью следующей таблицы:

**Ценность самоцветов**

| d20   | Ценность самоцвета |
| ----- | ------------------ |
| 1–4   | <Coin v="10" />    |
| 5–9   | <Coin v="50" />    |
| 10–15 | <Coin v="100" />   |
| 16–19 | <Coin v="500" />   |
| 20    | <Coin v="1000" />  |

## Драгоценности

Каждая драгоценность стоит 3d6 × <Coin v="100" /> (при желании [рефери](/adventures/running-adventures/the-referees-role) может увеличить это значение для игроков с уровнем выше 3-го).

### Сломанные драгоценности

Обращение с драгоценностями ненадлежащим образом (например, сильные удары по ним, воздействие высоких температур или электричества) снижает их ценность на 50%.

## Суммарная ценность

Если, согласно описанию типа сокровищ, в кладе должно быть несколько самоцветов, рефери может комбинировать их, как считает нужным, главное, чтобы суммарная ценность оставалась прежней. То же относится и к драгоценностям. Например, в результате случайного выбора выяснилось, что в кладе есть 5 самоцветов, каждый из которых стоит <Coin v="50" />. В таком случае их можно заменить одним драгоценным камнем стоимостью <Coin v="250" /> или подобрать любую другую комбинацию (например, один самоцвет за <Coin v="100" /> и три других по <Coin v="50" /> каждый).
